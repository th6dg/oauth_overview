### AI usage practices for developers
#### Todo list
- Install Copilot and enable mission control in VS Code. Run your first workflow (“explain repo + list failing tests”).
- Connect to GitHub MCP server and use agent mode to add a small feature or test.
- Use mission control to generate tests or scaffold a small feature.
- Enable Copilot code review and open a pull request.
- Assign a refactor issue to the Copilot coding agent.
- Evaluate the results and iterate as needed.
#### Goals
- Gain hands-on experience using AI tools like GitHub Copilot for coding, testing, and code review.
#### Resources
- GitHub Copilot documentation: https://docs.github.com/en/copilot
